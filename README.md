# SQL Data Warehouse Project

## Project Overview

This project demonstrates the design and analysis of a Retail Sales Data Warehouse using MySQL.

The project includes:

* Database design
* Table creation using Primary and Foreign Keys
* Data insertion and management
* Business analytics queries
* JOIN operations
* Common Table Expressions (CTEs)
* Window Functions

---

## Database Schema

### Customers

Stores customer information.

### Products

Stores product details, categories, and pricing.

### Orders

Stores order-level transaction information.

### Order_Details

Stores product-level sales transactions for each order.

---

## Business Questions Solved

1. Total Revenue Generated
2. Revenue by Customer
3. Top Selling Products
4. Revenue by Category
5. Customer Ranking by Revenue
6. Customers with No Orders
7. Monthly Sales Trend
8. High Revenue Customers using CTEs
9. Top Product in Each Category

---

## Key Insights

* The highest revenue was generated by Neel.
* Electronics contributed the largest share of total revenue.
* Laptops were the highest revenue-generating product.
* Window Functions were used to rank customers based on spending.
* CTEs helped simplify analytical queries and improve readability.
